# Sales Data Analysis - Python

This repository contains a **simple Python project** to analyze sales data using **pandas**.  
It demonstrates reading CSV files, calculating revenue, identifying top-selling products, and summarizing sales over time.


## Features
- Load sales data from a CSV file (`sales_data.csv`)  
- Calculate total revenue  
- Find top-selling products  
- Analyze sales trends over time (monthly)  
- Works with a sample dataset provided in the repo 


## Files
1. `create_sales_csv.py` – Generates a sample `sales_data.csv` file for testing  
2. `sales_analysis.py` – Main Python script to analyze the sales data using pandas
